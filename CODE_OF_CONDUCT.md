# Code of Conduct

## 1. Our Commitment

The **Chimera project** is an open-source school group project focused on building an AI system that translates hand signs into text.

We aim to create a **safe, respectful, and collaborative learning environment** where all contributors can improve their skills in programming, AI, and teamwork.

---

## 2. Our Values

In this project, we value:

* Collaboration over competition
* Learning over perfection
* Curiosity and experimentation
* Respect for different skill levels
* Shared growth as developers

---

## 3. Expected Behavior

All contributors are expected to:

* Be respectful and kind in discussions
* Help others learn and improve
* Give constructive feedback, not criticism
* Be open to ideas and suggestions
* Communicate clearly and responsibly

---

## 4. Unacceptable Behavior

To maintain a healthy learning environment, the following are discouraged:

* Disrespectful or offensive language
* Personal attacks or toxic behavior
* Deliberate disruption of the project
* Ignoring feedback in a harmful or uncooperative way
* Any form of harassment

---

## 5. Learning Environment

Since this is a **school-based project**, mistakes are expected and part of the process.

* Code reviews are meant for learning, not judging
* Questions are always welcome
* No one is expected to be perfect
* Everyone contributes at their own pace

---

## 6. Scope

This Code of Conduct applies to:

* GitHub repositories (issues, pull requests, commits)
* Project discussions and documentation
* Group collaboration and communication channels

---

## 7. Reporting Issues

If any behavior makes you uncomfortable or goes against this Code of Conduct, you may report it by:

* Contacting the project lead or group members privately
* Using GitHub Issues if appropriate

All concerns will be handled respectfully and within the group.

---

## 8. Our Standard

We believe a good project is not just about code, but about how people work together:

> A strong team builds strong systems.
