# Contributing to Chimera

Thank you for your interest in contributing to **Chimera** — a school open-source project that translates hand signs into text.

We welcome contributions from everyone, especially learners who want to grow in **computer vision, and software development**.

---

## Getting Started

Before contributing, please make sure you:

* Read the [README](../README.md)
* Understand the project goal and structure
* Familiarize yourself with the codebase
* Follow the Code of Conduct

---

## How to Contribute

You can contribute to this project in the following ways:

- Check your assigned tasks here: 👉 [View Member Tasks](./docs/members.md)
- Improve existing features or fix bugs
- Help improve documentation
- Test the system and report issues


---

## 🛠️ Development Workflow

1. Clone the repository
2. Create a new branch

   ```bash
   git checkout -b <name>
   ```
3. Make your changes
4. Commit your changes:
    We create a CLI tool for out commit convention so read this [convention](./docs/dev/convention.md)
5. Push to your fork

   ```bash
   git push origin <name>
   ```
6. Open a Pull Request

---

## 📏 Guidelines

Please follow these rules when contributing:

* Keep code clean and readable
* Write meaningful commit messages
* Avoid large unrelated changes in one PR
* Be respectful during discussions
* Focus on learning and collaboration

---

## 💬 Communication

If you have ideas, suggestions, or questions:

* Open a GitHub Issue
* Discuss improvements in Pull Requests
* Talk with the team respectfully

---

## 🎓 Note for Contributors

This project is part of a **school-based learning environment**, so:

* Mistakes are expected and encouraged as part of learning
* Feedback is meant to help you improve
* Everyone is learning at different levels

---

## 🙌 Thank You

Your contributions help improve **Chimera** and make it a better learning project for everyone involved.
